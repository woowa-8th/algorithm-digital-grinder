# algorithm-digital-grinder

## 목표
네카라쿠배당토직야 코테 씹어먹기

## 규칙
1. 매주에 정해진 문제를 하루에 하나씩 풀이
    - 템플릿에 맞춰서 풀이
    - 제출 코드는 `사이트` &rarr; `문제 번호` 폴더를 생성하여 커밋
2. 매주 월요일 오후 6시 이후에 전주 풀이 발표와 이번주 개념 설명을 진행

## Convention
### Pull Request
- `[N주차] 이름`
- 예시) [1주차] 서여

### Commit
- `이름: 문제 번호 문제 이름`
- 예시) 서여: 1001 A-B

### File Name
- `이니셜_문제 번호.java`
- 예시) SY_1001.java
